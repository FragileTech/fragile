### Causal Set (Bombelli et al. 1987)

**Type:** Definition
**Label:** `def-causal-set-standard`
**Source:** [13_fractal_set_new/11_causal_sets.md](13_fractal_set_new/11_causal_sets.md)
**Tags:** `causal-set`, `fractal-set`

**Statement:**
A **causal set** $(C, \prec)$ is a locally finite partially ordered set (poset) satisfying:

**Axiom CS1 (Irreflexivity)**: For all $e \in C$, $e \not\prec e$

**Axiom CS2 (Transitivity)**: For all $e_1, e_2, e_3 \in C$, if $e_1 \prec e_2$ and $e_2 \prec e_3$, then $e_1 \prec e_3$

**Axiom CS3 (Local Finiteness)**: For all $e_1, e_2 \in C$, the set $\{e \in C : e_1 \prec e \prec e_2\}$ is finite

**Physical interpretation**:
- Elements $e \in C$ represent spacetime events
- $e_1 \prec e_2$ means "$e_1$ causally precedes $e_2$" (future light cone)
- Local finiteness = finite number of events in any causal interval

### Poisson Sprinkling

**Type:** Definition
**Label:** `def-poisson-sprinkling`
**Source:** [13_fractal_set_new/11_causal_sets.md](13_fractal_set_new/11_causal_sets.md)
**Tags:** `causal-set`, `fractal-set`, `poisson-sprinkling`

**Statement:**
Given a Lorentzian manifold $(M, g_{\mu\nu})$ with volume element $dV = \sqrt{-\det g} \, d^d x$, a **Poisson sprinkling** with density $\rho$ is:

1. **Sample points**: Draw $N$ points $\{x_i\}$ from $M$ with probability density $p(x) = \rho \cdot \sqrt{-\det g(x)} / V_{\text{total}}$

2. **Define order**: $e_i \prec e_j$ iff $x_i$ is in the causal past of $x_j$ (i.e., timelike/lightlike separated with $t_i < t_j$)

**Property**: For constant density $\rho = \text{const}$, expected number of elements in causal interval $I(e_1, e_2)$ is $\mathbb{E}[|I(e_1, e_2)|] = \rho \cdot V(I)$ where $V(I)$ is Lorentzian volume.
