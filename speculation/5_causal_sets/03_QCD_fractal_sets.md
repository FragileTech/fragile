## 2. Wilson action on irregular cycles

:::{admonition} üîó From Loops to Forces
:class: important
The **Wilson action** turns loop measurements into dynamics:
- Small loops measure the local field strength (like electromagnetic field)
- The action sums over all fundamental loops with appropriate weights
- Minimizing the action determines the gauge field configuration
- This discrete action converges to the continuous Yang-Mills action!

Think of it as springs: each loop is a spring, and the system finds the configuration that minimizes total spring energy.
:::

### 2.1 Gauge‚Äëinvariant action

:::{prf:definition} Irregular Wilson action
:label: def-wilson-action-qcd
:nonumber:

Fix a local normalization scale $\varepsilon$ and weights $w_\varepsilon(C)>0$. Define the gauge action

$$
\boxed{\;S_g^{(\varepsilon)}[U]\;=\;\frac{\beta_\varepsilon}{2N_c}\sum_{C\in\mathcal C_{\mathrm{fund}}}\;w_\varepsilon(C)\,\Big(1-\frac{1}{N_c}\mathrm{Re}\,\mathrm{Tr}\,\mathcal U(C)\Big)\;}
$$

with $N_c=3$ and $\beta_\varepsilon=2N_c/g_\varepsilon^2$.
:::

:::{prf:theorem} Gauge invariance
:label: thm-gauge-invariance-qcd
:nonumber:

$S_g^{(\varepsilon)}[U]$ is invariant under $U_{e\to e'}\mapsto g(e)U_{e\to e'}g(e')^\dagger$.
:::
:::{prf:proof}
:label: proof-gauge-invariance-qcd

Each $\mathcal U(C)$ transforms by conjugation with the group element at the basepoint; $\mathrm{Tr}$ is conjugacy‚Äëinvariant. Weights and the constant term are gauge‚Äëblind.
:::

### 2.3 Strong‚Äëcoupling confinement on the graph
