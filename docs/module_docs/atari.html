AtariEnv¶

class fragile.atari.env.AtariEnv(env: plangym.env.Environment)[source]¶

The AtariEnv acts as an interface with plangym.AtariEnvironment.

It can interact with any Atari environment that follows the interface of plangym.

class fragile.atari.walkers.AtariWalkers(max_reward: int = None, *args, **kwargs)[source]¶

This Walkers incorporate an additional stopping mechanism for the walkers that allows to set a maximum score, and finish if the a given game has been completely cleared.

calculate_end_condition() → bool[source]¶

Process data from the current state to decide if the iteration process should stop. It not only keeps track of the maximum number of iterations and the death condition, but also keeps track if the game has been played until it finished.

Returns

Boolean indicating if the iteration process should be finished. True means it should be stopped, and False means it should continue.
