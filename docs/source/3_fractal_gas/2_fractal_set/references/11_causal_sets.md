## 1. Causal Set Theory: Axioms and Framework

### 1.1. Standard Causal Set Definition

:::{prf:definition} Causal Set (Bombelli et al. 1987)
:label: def-causal-set-standard

A **causal set** $(C, \prec)$ is a locally finite partially ordered set (poset) satisfying:

**Axiom CS1 (Irreflexivity)**: For all $e \in C$, $e \not\prec e$

**Axiom CS2 (Transitivity)**: For all $e_1, e_2, e_3 \in C$, if $e_1 \prec e_2$ and $e_2 \prec e_3$, then $e_1 \prec e_3$

**Axiom CS3 (Local Finiteness)**: For all $e_1, e_2 \in C$, the set $\{e \in C : e_1 \prec e \prec e_2\}$ is finite

**Physical interpretation**:
- Elements $e \in C$ represent spacetime events
- $e_1 \prec e_2$ means "$e_1$ causally precedes $e_2$" (future light cone)
- Local finiteness = finite number of events in any causal interval
:::

### 1.2. Poisson Sprinkling (Standard Construction)

:::{prf:definition} Poisson Sprinkling
:label: def-poisson-sprinkling

Given a Lorentzian manifold $(M, g_{\mu\nu})$ with volume element $dV = \sqrt{-\det g} \, d^d x$, a **Poisson sprinkling** with density $\rho$ is:

1. **Sample points**: Draw $N$ points $\{x_i\}$ from $M$ with probability density $p(x) = \rho \cdot \sqrt{-\det g(x)} / V_{\text{total}}$

2. **Define order**: $e_i \prec e_j$ iff $x_i$ is in the causal past of $x_j$ (i.e., timelike/lightlike separated with $t_i < t_j$)

**Property**: For constant density $\rho = \text{const}$, expected number of elements in causal interval $I(e_1, e_2)$ is $\mathbb{E}[|I(e_1, e_2)|] = \rho \cdot V(I)$ where $V(I)$ is Lorentzian volume.
:::

**Limitation of Poisson sprinkling**: Uniform density $\rho = \text{const}$ does not adapt to local geometry, leading to:
- Over-sampling in flat regions (wasteful)
- Under-sampling in curved regions (loss of information)

## 3. Proof that Fractal Set Satisfies Causal Set Axioms

### 3.1. Causal Order on Episodes

:::{prf:definition} Causal Order on Fractal Set
:label: def-fractal-set-causal-order

For episodes $e_i, e_j \in E$ with positions $x_i, x_j \in \mathcal{X}$ and times $t_i, t_j \in \mathbb{R}$, define:

$$
e_i \prec_{\text{CST}} e_j \quad \iff \quad t_i < t_j \text{ and } d_{\mathcal{X}}(x_i, x_j) < c(t_j - t_i)
$$

where:
- $d_{\mathcal{X}}(x_i, x_j)$ is the **Riemannian distance** on $(\mathcal{X}, g)$
- $c$ is the effective "speed of light" (maximal information propagation speed)

**Physical interpretation**: $e_i \prec e_j$ means "information from episode $e_i$ can causally influence episode $e_j$"
:::

:::{prf:remark} Relation to Lorentzian Structure
:label: rem-lorentzian-from-riemannian

The emergent Riemannian metric $g(x)$ on $\mathcal{X}$ can be promoted to a Lorentzian metric on spacetime $M = \mathbb{R} \times \mathcal{X}$ by:

$$
ds^2 = -c^2 dt^2 + g_{ij}(x) dx^i dx^j
$$

The causal order $\prec_{\text{CST}}$ then corresponds to the **chronological order** induced by this Lorentzian metric:

$$
e_i \prec e_j \quad \iff \quad x_j \in J^+(x_i)
$$

where $J^+(x_i)$ is the causal future of event $x_i$ in $(M, g_{\mu\nu})$.

### 3.2. Verification of Causal Set Axioms

:::{prf:theorem} Fractal Set is a Valid Causal Set
:label: thm-fractal-set-is-causal-set

The Fractal Set $\mathcal{F} = (E, \prec_{\text{CST}})$ satisfies all causal set axioms.
:::

:::{prf:proof}
We verify each axiom:

**Axiom CS1 (Irreflexivity): $e \not\prec e$ for all $e \in E$**

For any episode $e_i$ with time $t_i$:

$$
e_i \prec_{\text{CST}} e_i \quad \iff \quad t_i < t_i \text{ and } d_{\mathcal{X}}(x_i, x_i) < c(t_i - t_i)
$$

$$
\iff \quad t_i < t_i \text{ and } 0 < 0
$$

Both conditions are false, so $e_i \not\prec_{\text{CST}} e_i$. âœ“

**Axiom CS2 (Transitivity): $e_1 \prec e_2$ and $e_2 \prec e_3$ implies $e_1 \prec e_3$**

Assume $e_1 \prec_{\text{CST}} e_2$ and $e_2 \prec_{\text{CST}} e_3$. Then:

1. $t_1 < t_2 < t_3$ (time ordering is transitive)
2. $d_{\mathcal{X}}(x_1, x_2) < c(t_2 - t_1)$
3. $d_{\mathcal{X}}(x_2, x_3) < c(t_3 - t_2)$

By the **triangle inequality** for Riemannian distance:

$$
d_{\mathcal{X}}(x_1, x_3) \leq d_{\mathcal{X}}(x_1, x_2) + d_{\mathcal{X}}(x_2, x_3)
$$

$$
< c(t_2 - t_1) + c(t_3 - t_2) = c(t_3 - t_1)
$$

Therefore $t_1 < t_3$ and $d_{\mathcal{X}}(x_1, x_3) < c(t_3 - t_1)$, which means $e_1 \prec_{\text{CST}} e_3$. âœ“

**Axiom CS3 (Local Finiteness): $|\{e \in E : e_1 \prec e \prec e_2\}| < \infty$**

The set $I(e_1, e_2) := \{e \in E : e_1 \prec e \prec e_2\}$ is the **causal interval** between $e_1$ and $e_2$.

**Geometric characterization**: An episode $e$ is in $I(e_1, e_2)$ iff:

$$
t_1 < t < t_2 \quad \text{and} \quad d_{\mathcal{X}}(x_1, x) < c(t - t_1) \quad \text{and} \quad d_{\mathcal{X}}(x, x_2) < c(t_2 - t)
$$

This defines a **bounded region** $\mathcal{D} \subset \mathbb{R} \times \mathcal{X}$ (a "double cone" in spacetime).

**Expected number of episodes in $\mathcal{D}$**:

$$
\mathbb{E}[|I(e_1, e_2)|] = \int_{\mathcal{D}} \rho_{\text{episode}}(t, x) \, dt \, dx
$$

$$
= \int_{t_1}^{t_2} dt \int_{\mathcal{B}_t} \rho_{\text{spatial}}(x) \sqrt{\det g(x)} \, dx
$$

where $\mathcal{B}_t = \{x : d_{\mathcal{X}}(x_1, x) < c(t - t_1) \text{ and } d_{\mathcal{X}}(x, x_2) < c(t_2 - t)\}$ is the spatial slice at time $t$.

**Key observations**:
1. $\mathcal{B}_t$ is a **compact subset** of $\mathcal{X}$ (bounded by two balls)
2. $\rho_{\text{spatial}}(x) = (1/Z) \sqrt{\det g} e^{-U_{\text{eff}}/T}$ is **integrable** over compact sets
3. The time integral is over a **finite interval** $[t_1, t_2]$

Therefore $\mathbb{E}[|I(e_1, e_2)|] < \infty$, and for any finite realization, $|I(e_1, e_2)| < \infty$ almost surely. âœ“

**Conclusion**: All three axioms are satisfied, so $(E, \prec_{\text{CST}})$ is a causal set.

**Q.E.D.** $\square$

## 4. Faithful Discretization and Manifoldlikeness

### 4.1. Faithful Embedding Theorem

:::{prf:theorem} Fractal Set is Faithful Discretization of Riemannian Manifold
:label: thm-fractal-set-faithful-embedding

Let $(\mathcal{X}, g)$ be the emergent Riemannian manifold with metric $g(x) = H(x) + \epsilon_\Sigma I$. The Fractal Set $(E, \prec_{\text{CST}})$ provides a **faithful discretization** in the sense:

1. **Volume Matching**: The number of episodes in a region $\Omega \subset \mathcal{X}$ satisfies:

$$
\mathbb{E}\left[\frac{|E \cap \Omega|}{N}\right] = \frac{1}{Z} \int_{\Omega} \sqrt{\det g(x)} e^{-U_{\text{eff}}/T} dx + O(N^{-1/2})
$$

2. **Metric Recovery**: The Riemannian distance can be recovered from the causal structure via:

$$
d_{\mathcal{X}}(x_i, x_j) = c \cdot \inf\{t_j - t_i : e_i \prec e \prec e_j \text{ with } t_e = t_i + t/c\}
$$

3. **Dimension Estimation**: The Myrheim-Meyer dimension estimator converges:

$$
d_{\text{MM}} := \lim_{N \to \infty} \frac{\ln \mathbb{E}[|I(e_1, e_2)|]}{\ln \mathbb{E}[\sqrt{|I(e_1, e_3)| \cdot |I(e_3, e_2)|}]} \xrightarrow{N \to \infty} d
$$

where $d = \dim \mathcal{X}$ is the manifold dimension.
:::

:::{prf:proof}
**Part 1 (Volume Matching)**:

The expected number of episodes in $\Omega$ is:

$$
\mathbb{E}[|E \cap \Omega|] = N \cdot \mathbb{P}(e \in \Omega) = N \int_{\Omega} \frac{\rho_{\text{spatial}}(x)}{Z} dx
$$

$$
= \frac{N}{Z} \int_{\Omega} \sqrt{\det g(x)} e^{-U_{\text{eff}}/T} dx
$$

Normalizing by $N$ gives the stated result. The $O(N^{-1/2})$ error comes from finite-sample fluctuations (Central Limit Theorem).

**Part 2 (Metric Recovery)**:

The Riemannian distance $d_{\mathcal{X}}(x_i, x_j)$ is the length of the shortest path in $(\mathcal{X}, g)$.

For episodes $e_i$ at $(t_i, x_i)$ and $e_j$ at $(t_j, x_j)$, consider a chain $e_i \prec e \prec e_j$ where $e$ is at $(t, x)$.

From the causal order definition:
- $d_{\mathcal{X}}(x_i, x) < c(t - t_i)$
- $d_{\mathcal{X}}(x, x_j) < c(t_j - t)$

The shortest such chain gives $d_{\mathcal{X}}(x_i, x_j) \approx c(t_j - t_i)$ for direct causal connection.

For non-direct connections, the inf over chains approximates the geodesic distance.

**Part 3 (Dimension Estimation)**:

The Myrheim-Meyer dimension estimator uses the scaling of causal intervals.

For a $d$-dimensional manifold with Lorentzian metric, the expected size of a causal interval scales as:

$$
\mathbb{E}[|I(e_1, e_2)|] \sim (\Delta t)^d
$$

where $\Delta t = t_2 - t_1$ is the temporal separation.

The estimator extracts $d$ from the scaling exponent. For the Fractal Set:

$$
\mathbb{E}[|I(e_1, e_2)|] = \int_{t_1}^{t_2} dt \int_{\mathcal{B}_t} \rho_{\text{spatial}}(x) \sqrt{\det g} \, dx \sim (t_2 - t_1)^d
$$

for small $t_2 - t_1$. Thus $d_{\text{MM}} \to d$ as $N \to \infty$.

**Q.E.D.** $\square$
:::

### 4.2. Comparison with Poisson Sprinkling

:::{prf:proposition} Fractal Set Improves on Poisson Sprinkling
:label: prop-fractal-vs-poisson

Compared to uniform Poisson sprinkling with density $\rho_0 = N/V_{\text{total}}$, the Fractal Set with adaptive density $\rho(x) = \sqrt{\det g(x)} \psi(x)$ achieves:

1. **Better geometric fidelity**: Variance in local sample count is:

$$
\text{Var}_{\text{Fractal}}[|E \cap \Omega|] < \text{Var}_{\text{Poisson}}[|E \cap \Omega|]
$$

for regions $\Omega$ where $\sqrt{\det g}$ varies significantly.

2. **Optimal information content**: The Kullback-Leibler divergence from the true volume measure is minimized:

$$
D_{\text{KL}}(\rho_{\text{true}} \| \rho_{\text{Fractal}}) < D_{\text{KL}}(\rho_{\text{true}} \| \rho_{\text{Poisson}})
$$

where $\rho_{\text{true}} = \sqrt{\det g} / V_{\text{total}}$.
:::

:::{prf:proof}
**Part 1 (Geometric Fidelity)**:

For Poisson sprinkling with uniform density $\rho_0$:

$$
\text{Var}_{\text{Poisson}}[|E \cap \Omega|] = \rho_0 \int_{\Omega} \sqrt{\det g} \, dx
$$

For Fractal Set with adaptive density:

$$
\text{Var}_{\text{Fractal}}[|E \cap \Omega|] = \int_{\Omega} \rho(x) [1 - \rho(x)] dx \leq \int_{\Omega} \rho(x) dx
$$

When $\rho(x) \propto \sqrt{\det g(x)}$ matches the local geometry, the variance is reduced because sampling is concentrated where the volume element is large.

**Part 2 (Information Content)**:

The KL divergence from the true volume measure $\rho_{\text{true}} = \sqrt{\det g} / V$ to the Fractal Set density is:

$$
D_{\text{KL}}(\rho_{\text{true}} \| \rho_{\text{Fractal}}) = \int \rho_{\text{true}} \log \frac{\rho_{\text{true}}}{\rho_{\text{Fractal}}} dx
$$

$$
= \int \frac{\sqrt{\det g}}{V} \log \frac{\sqrt{\det g} / V}{\sqrt{\det g} \psi / Z} dx = \log(Z/V) + \int \rho_{\text{true}} \log(1/\psi) dx
$$

For Poisson with constant density $\rho_0 = N/V$:

$$
D_{\text{KL}}(\rho_{\text{true}} \| \rho_{\text{Poisson}}) = \int \frac{\sqrt{\det g}}{V} \log \frac{\sqrt{\det g}}{\rho_0} dx
$$

When $\sqrt{\det g}$ varies spatially, the constant $\rho_0$ cannot match it everywhere, leading to higher KL divergence.

The Fractal Set with $\rho \propto \sqrt{\det g}$ achieves the minimum KL divergence (modulo the fitness factor $\psi$).

**Q.E.D.** $\square$

- $I^+(e) = \{e' : e \prec e'\}$ is the causal future
