SECTION SUMMARY
================================================================================

Section 11 establishes the continuity properties of the distance-to-companion
measurement operator through a two-stage analysis:

PART 1: Continuity of Expected Raw Distance Vector (k ≥ 2 regime)
- Decomposes error into positional and structural components
- Separates analysis for stable walkers (status unchanged) and unstable 
  walkers (status changed)
- Derives deterministic bound with explicit coefficients
- Identifies singularity at k=1

PART 2: Mean-Square Continuity of Sampled Raw Distance Vector
- Proves the operator satisfies the Bounded Variance Axiom
- Combines variance bound with expected value bound
- Derives explicit bound function F_{d,ms} for standardization operator analysis

KEY MATHEMATICAL INSIGHTS:

1. Error Decomposition Strategy:
   - Positional error: Due to walker displacement
   - Structural error: Due to companion set changes
   - Own-status error: Due to walker's status change

2. Singularity at k=1:
   - Coefficient C_{status,d}^{(2)}(k_1) ~ 1/(k_1-1)^2
   - Reflects fundamental discontinuity in diversity measurement
   - When k=1, expected distance vector becomes deterministically zero

3. Variance Analysis:
   - Uniform bound κ²_variance = N·D_Y²
   - Independent of swarm configuration
   - Critical for mean-square continuity

4. Technical Tools:
   - Reverse triangle inequality for metric spaces
   - Jensen's inequality for expectation
   - Double summation analysis for companion averaging
   - Elementary inequality (a+b)² ≤ 2a² + 2b²

KEY RESULTS
================================================================================

Main Theorem (thm-distance-operator-mean-square-continuity):
  E[||d(S₁) - d(S₂)||₂²] ≤ F_{d,ms}(S₁, S₂)

where:
  F_{d,ms} = 6N·D_Y² + 3(C_{pos,d}·Δ_{pos}² + C_{status,d}^{(1)}·n_c + C_{status,d}^{(2)}(k₁)·n_c²)

with coefficients:
  C_{pos,d} = 12
  C_{status,d}^{(1)} = D_Y²
  C_{status,d}^{(2)}(k₁) = 8k₁D_Y²/(k₁-1)²

Discontinuity Result (thm-expected-raw-distance-k1):
  |A(S)| = 1 ⟹ E[d(S)] = 0

This provides the critical input for the standardization operator continuity
analysis in subsequent sections.
